A BlackJack application complete with a dynamic GUI as well as music.
To ran the program run the file BlackJack.java.
There is an additional text tracking of the game in the compiler.

Inside the zip file should be 8 files (including this readme).

The files contained in the zip:
Deck: Contains all of the card assets, along with the background image and the music file
BlackJack: Java file (contains the main method)
UserInterface: Java file
GameMath: Java file with game logic
Sound: Java file containing code for the in game music
Card: Java file
BackgroundImage: Java file
